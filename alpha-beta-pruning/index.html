Alpha-Beta Pruning

Alpha-Beta Pruning is an optimization technique for the minimax algorithm that reduces the number of nodes evaluated in the search tree.

Alpha (α): Best value that the maximizer currently can guarantee at that level or above.

Beta (β): Best value that the minimizer currently can guarantee at that level or above.

Pruning occurs when a node's value cannot possibly influence the final decision.

Instructions:

  • Click on a node to select it
  • Enter a value and click "Set Value"
  • Use Next Step or Autoplay to run the algorithm
