Depth First Search

Depth First Search (DFS) is an algorithm that explores a graph by going as deep as possible along each branch before backtracking.

How DFS Works: The algorithm starts at the root node and explores as far as possible along each branch before backtracking. It uses a stack to remember which nodes to visit next.

Why a Node is Chosen: DFS always chooses the most recently discovered unvisited node to explore next, following the "last in, first out" (LIFO) principle.
